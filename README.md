# 📅 日程任务管理系统

一个简单高效的日程任务管理 Web 应用，基于 Streamlit 开发。

## ✨ 功能特性

### 📋 任务管理
- ✅ 添加、编辑、删除任务
- 📁 分类管理（工作/生活/游戏运营/学习/其他）
- 🔴🟡🟢 优先级设置（高/中/低）
- 📅 截止日期提醒
- ✔️ 完成状态跟踪

### 📅 日程管理
- 📆 日历视图
- ⏰ 时间设置
- 🔄 重复事件（每天/每周/每月）
- 📝 描述备注

### 📊 数据统计
- 📈 任务完成率
- 📊 分类分布图
- 📉 日程时间线

### 💾 数据管理
- 导出 JSON/CSV
- 本地 SQLite 存储
- 数据安全私密

## 🚀 快速启动

### 方式 1：双击启动
```bash
双击 start.bat 文件
```

### 方式 2：命令行启动
```bash
cd C:\Users\11348\.qclaw\workspace\task-manager
streamlit run app.py
```

启动后，浏览器会自动打开：http://localhost:8501

## 📱 使用说明

### 首页
- 查看任务统计概览
- 快速添加任务/日程
- 查看今日待办

### 任务管理
- 点击"✅ 任务管理"进入
- 使用筛选器按分类/状态/优先级筛选
- 勾选复选框标记完成
- 点击🗑️删除任务

### 日程管理
- 点击"📅 日程管理"进入
- 选择日期范围查看日程
- 支持设置重复事件

### 数据统计
- 查看任务完成率
- 分类分布图表
- 日程时间分布

### 设置
- 导出数据（JSON/CSV）
- 清空数据（谨慎操作）

## 🌐 多设备访问

### 本机访问
```
http://localhost:8501
```

### 局域网访问（手机/其他电脑）
1. 查看本机 IP 地址：
   ```bash
   ipconfig
   ```
   找到 IPv4 地址，例如：192.168.1.100

2. 启动时指定地址：
   ```bash
   streamlit run app.py --server.address 0.0.0.0
   ```

3. 其他设备访问：
   ```
   http://192.168.1.100:8501
   ```

- **后端**：Python
- **数据库**：SQLite（本地存储）
- **数据处理**：Pandas
